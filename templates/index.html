Integration

2 lines of code.
Full AI protection.

Integrate into any Python, Node.js or any language app in under 60 seconds. No infrastructure changes.

1
Get your API key
Register free, get your sp_live_... key instantly.
2
Send the prompt
POST any user prompt to our API before passing it to your LLM.
3
Block or allow
We return SAFE or MALICIOUS with confidence score in <50ms.
your_app.py
import requests

# Send prompt to ShieldPrompt first
res = requests.post(
  "https://api.shieldprompt.in/v1/check",
  headers={"X-API-Key": "sp_live_..."},
  json={"prompt": user_input}
).json()

if res["is_malicious"]:
  return "Request blocked 🛡️"

# Safe — pass to your LLM
call_your_llm(user_input)
